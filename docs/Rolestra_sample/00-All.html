AI 직원들과
함께 일하는 사무실.

Rolestra는 Claude Code / Gemini / Codex 같은 AI 에이전트를 직원처럼 고용하고, 메신저로 지시하고, 결재로 권한을 조절하는 AI 사무실 OS입니다. 아래는 6개 핵심 화면의 디자인 탐색 — 3가지 테마(Warm / Tactical / Retro) × 2가지 모드(Light / Dark).

6 화면 36 변형 (6×6) 3 테마 1280×820 아트보드
./screens
01
01 · 사무실 대시보드
진행 중인 프로젝트 · 결재 대기 · 직원 상태 · 최근 대화를 한 화면에. 위젯 박스(DashCard)가 모든 화면의 기본 형태로 사용됩니다.
홈 위젯 그리드 SSM 12단계
02
02 · 메신저 스레드
대표(사람)와 AI 직원들 사이의 실시간 대화. 결재 버튼 · 합의 지표 · SSM 스테이지가 메시지 안에 그대로 녹아 있습니다.
실시간 합의 지표 인라인 결재
03
03 · 결재함
쌓인 결재 요청을 한 번에. 좌측 큐 + 우측 상세(디프 · 변경 사유 · 영향 범위) + 한 번의 키로 허가 · 조건부 · 거절.
결재 디프 뷰어 키보드 우선
04
04 · 작업 큐
대표가 내린 지시사항이 쌓이고 한 건씩 실행. 상단 통계 · 현재 작업 스포트라이트 · 완료 / 진행 / 대기 / 실패 전체 테이블.
큐 실행 현황 타임라인
05
05 · 설정
조직 · 직원 · 권한 정책 · 외부 연결 관리. 기본 권한 모드 (auto / hybrid / manual)와 SSM 마감 단계를 테이블 + 폼으로 조정.
어드민 권한 정책 직원 관리
06
06 · 온보딩
5단계 마법사 중 "직원 선택" 단계. 설치된 CLI를 자동 감지해서 입사 후보로 제시. 체크해서 고용 → 역할 부여 → 권한 설정 → 첫 프로젝트.
