## V4 / R12+ 이연 항목

R11 종료 후 차기 메이저 (V4) 또는 출시 후 인프라/보안 (R12+):

### V4 (사용자 출시 후)

- DM read-receipt / typing indicator 실 이벤트
- 파일 첨부 드래그앤드롭
- 음성 메모
- 플러그인 시스템
- ComfyUI / SD 연동 (이미지 생성)
- 회의별 LLM 요약 provider drop-down
- MessageSearchView 사이드 패널 layout (R10 D2)
- Onboarding step 3/4/5 시안 풍부화 (시안 06 추가 캡처 대기)
- design polish 라운드 3+ — Hero strip 통합 (G5) / InsightStrip footer 변경 (G4) / Queue 6-column 테이블 (Q3)

### R12+ (출시 후 인프라/보안)

- macOS 코드 사인 + 공증 (notarytool)
- Windows 코드 사인 (EV cert)
- AutoUpdate (electron-updater)
- Remote Access v3 재설계 (R10 Known Concern #8 의 remote-* 테스트가 skip)
- Memory Phase 3-b — 임베딩 + 하이브리드 검색 + 반성 + 진화 (Stanford 3-factor scoring)
- Sentry / 크래시 리포팅 (현재 로컬만)
- Localization 추가 (ja / zh-CN)

### 각하

- Settings vertical sidebar nav (R10 D3 — horizontal Radix Tabs 유지)
- DM 별도 `dm_sessions` 테이블 (R10 D1 — `channels.kind='dm'` 유지)
